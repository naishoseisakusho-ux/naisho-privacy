プライバシーポリシー

アプリ「ナイショ秘書」 / 最終更新日: 2026年7月19日

本プライバシーポリシーは、アプリ「ナイショ秘書」（以下「本アプリ」）における情報の取り扱いについて定めるものです。

本アプリのAI処理はすべて端末内で完結します。あなたが入力した文章が、当方のサーバーや第三者に送信されることはありません。

1. 基本方針 — 端末内で処理します

本アプリのAIによる文章処理（生成・翻訳・要約・校正・議事録作成など）は、すべてお使いの端末内で行われます。あなたが入力・貼り付けたテキスト（メール本文、文章、発言ログ等）を、当方のサーバーや第三者に送信・保存することはありません。

2. 当方が収集しない情報

本アプリは、AI機能の利用にあたり、入力された文章の内容を外部に送信・収集しません。作成された下書き、タスク、定型文は、端末内にのみ保存されます。

また、本アプリは以下の情報を一切収集しません。

  • 写真・動画・音声・ファイル
  • 連絡先・カレンダーの予定
  • 位置情報
  • 氏名・メールアドレス等の個人情報

3. 通信が発生する場面・収集する情報

本アプリは、次の目的で外部と通信する場合があります。

  • AIモデルのダウンロード: 初回などに、AIモデルの提供元（Hugging Face 等）からモデルファイルをダウンロードします。また、利用可能なモデル一覧を GitHub から取得します。各提供元の利用規約・プライバシーポリシーが適用されます。
  • アプリ内購入（課金）: 有料機能の購入時に、Google Play 課金（Google Play Billing）を利用します。決済情報は Google が処理し、当方がクレジットカード情報等を取得することはありません。
  • 利用状況データ（Firebase Analytics）: アプリの品質改善のため、匿名の利用状況（画面の表示、機能の選択、ボタン操作などの操作イベント、端末の機種・OSバージョン、アプリのバージョン）を Google（Firebase）へ送信する場合があります。入力・作成した文章の内容は一切含まれません。
  • クラッシュ診断データ（Firebase Crashlytics）: アプリの安定性向上のため、異常終了時に診断情報（端末の機種・OSバージョン、エラーの発生箇所やスタックトレース、アプリのバージョン、発生日時など）を Google（Firebase）へ送信します。入力・作成した文章の内容は含まれません。
  • プッシュ通知（Firebase Cloud Messaging）: お知らせの配信のため、Firebase Cloud Messaging を利用します。この際、端末を識別するための登録トークンが生成・利用されます。

4. 識別子について

上記の利用状況データ・クラッシュ診断・プッシュ通知にともない、個人を直接特定しない擬似的な識別子（Firebase のアプリインスタンスID、プッシュ通知の登録トークン）が利用されます。これらは氏名やメールアドレス等の個人情報とは紐づけていません。

5. 送信のオフ（オプトアウト）

利用状況データおよびクラッシュ診断データの送信は、アプリ内の「プライバシー設定」からいつでもオフにできます。オフにすると、以降これらのデータは送信されません。

プッシュ通知が不要な場合は、端末の通知設定から本アプリの通知を無効にできます。

6. 端末の権限について

本アプリが使用する権限は、以下のとおりです。

  • 通知: モデルのダウンロード進捗・完了、およびお知らせの通知のため
  • ネットワーク接続: AIモデルのダウンロード等のため

本アプリは、カメラ・マイク・連絡先・カレンダー・位置情報の権限を使用しません。

7. データの保存

作成した文章、タスク、定型文などは、お使いの端末内に保存されます。アプリを削除すると、これらの端末内データも削除されます（別途バックアップした場合を除く）。

8. データの削除について

  • 端末内のデータ（タスク・定型文・設定等）は、アプリの削除、または端末の設定からアプリのデータを消去することで削除されます。
  • 送信済みの匿名の利用状況・クラッシュ診断データの削除をご希望の場合は、第11項のお問い合わせ先までご連絡ください。

9. 第三者サービス

本アプリは、以下の第三者サービスを利用する場合があります。各サービスのプライバシーポリシーもご確認ください。

  • Google Play（配信・課金）
  • Firebase（Google）— 利用状況データ（Analytics）、クラッシュ診断（Crashlytics）、プッシュ通知（Cloud Messaging）
  • AIモデル提供元（Hugging Face 等）、モデル一覧の取得（GitHub）

10. 児童のプライバシー

本アプリは一般利用者（成人）を対象としており、児童を対象とした設計ではありません。児童から個人情報を収集することを意図していません。

11. 本ポリシーの変更 / お問い合わせ

本ポリシーは予告なく変更されることがあります。重要な変更がある場合は、アプリやストア上で告知します。
